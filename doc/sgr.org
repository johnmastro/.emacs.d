** Attribute codes

   |------+------------+------------|
   |   00 | None       |            |
   |   01 | Bold       |            |
   |   04 | Underscore |            |
   |   05 | Blink      |            |
   |   07 | Reverse    |            |
   |   08 | Concealed  |            |
   |------+------------+------------|
   |   30 | Black      | Foreground |
   |   31 | Red        | Foreground |
   |   32 | Green      | Foreground |
   |   33 | Yellow     | Foreground |
   |   34 | Blue       | Foreground |
   |   35 | Magenta    | Foreground |
   |   36 | Cyan       | Foreground |
   |   37 | White      | Foreground |
   |------+------------+------------|
   |   40 | Black      | Background |
   |   41 | Red        | Background |
   |   42 | Green      | Background |
   |   43 | Yellow     | Background |
   |   44 | Blue       | Background |
   |   45 | Magenta    | Background |
   |   46 | Cyan       | Background |
   |   47 | White      | Background |
